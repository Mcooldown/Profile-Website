Nginep Kuy

Nginep Kuy is the web application that made for BNCC Hackathon 4.0 with the theme about staycation. This application provide booking & checkout feature for accomodation, also Online Chat with Travel Agent feature to consult the best vacation plan that user can follows.

Link Web: https://nginepkuy.000webhostapp.com/
Link Github: https://github.com/Mcooldown/Hackathon2021_BNCC

This application is built with PHP Laravel 8 Framework, also with NPM to support the front-end. My job in this project mostly in back-end to create database, transform business logic into code, also help the other team members to optimize the front-end of the application.
