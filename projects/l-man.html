L-Man

L-Man is the Learning Management System for schools. This application is built to fulfilled the final project assignment of Software Engineering Course in Binus University. This application consist of several features, such as Forum Discussion, Daily Attendance, Class Schedule, Score, and Assignment. This application built with PHP Laravel 8 Framework.
