CentuDY

CentuDY is the web application for Online Medicine Shop. This application is built to fulfilled the final project assignment of Pattern Software Design Course in Binus University. This application consist of several features, such as Medicine Transaction and Transaction Report. This application built with ASP.NET Framework and DDD Concept
